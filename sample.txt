中文阅读器使用指南

欢迎使用中文阅读器！这是一个功能强大的阅读应用。

主要功能：
1. 书籍管理
   - 支持导入和保存 .txt 格式的电子书
   - 自动保存阅读进度和位置
   - 书架功能，支持分类和搜索

2. 阅读体验
   - 自动记忆上次阅读位置
   - 每30秒自动保存进度
   - 支持键盘快捷键导航
   - 支持全屏阅读模式

3. 笔记功能
   - 支持选中文本添加笔记
   - 笔记分类管理
   - 标签系统
   - 笔记导出功能

4. 界面定制
   - 支持明/暗主题切换
   - 字体大小调节
   - 字体族选择
   - 行高和页边距调整

快捷键说明：
- ← / → : 上一页/下一页
- Ctrl + B : 显示/隐藏书架
- Ctrl + N : 显示/隐藏笔记
- Ctrl + T : 切换主题
- F11 : 全屏模式

祝您阅读愉快！
